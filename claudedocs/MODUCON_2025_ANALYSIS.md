## 🚨 Critical Issues (치명적 문제)

### 1. 모바일 PWA 미구현 ⚠️⚠️⚠️
**심각도**: 🔴 Critical
**영향도**: 전체 프로젝트 목표 미달성

#### 문제 상황
- PRD 핵심 요구사항: "모바일 퍼스트 PWA"
- **현재 상태**: 기본 웹사이트만 있음
- Service Worker 없음
- manifest.json만 있고 실제 PWA 기능 없음
- 오프라인 모드 미구현

#### 영향
- 행사장 WiFi 불안정 시 사용 불가
- "Add to Home Screen" 불가
- 네이티브 앱 같은 UX 제공 불가
- PRD의 "설치 불필요" 약속 미이행

#### 해결 방안
```typescript
// 권장: next-pwa 재도입 (Static Export 호환 버전)
// next.config.ts
import withPWA from 'next-pwa';

export default withPWA({
  dest: 'public',
  disable: process.env.NODE_ENV === 'development',
  register: true,
  skipWaiting: true,
  runtimeCaching: [
    {
      urlPattern: /^https:\/\/api\.moducon\.vibemakers\.kr\/.*/i,
      handler: 'NetworkFirst',
      options: {
        cacheName: 'api-cache',
        expiration: {
          maxEntries: 32,
          maxAgeSeconds: 24 * 60 * 60 // 24 hours
        }
      }
    }
  ]
});
```
